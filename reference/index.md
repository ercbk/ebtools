## Time Series

- [`add_mase_scale_feat()`](https://ercbk.github.io/ebtools/reference/add_mase_scale_feat.md)
  : Add a scale feature by using a factor derived from the MASE error
  function

- [`add_tsfeatures()`](https://ercbk.github.io/ebtools/reference/add_tsfeatures.md)
  : Add time series features calculated by the tsfeatures package

- [`create_dtw_grids()`](https://ercbk.github.io/ebtools/reference/create_dtw_grids.md)
  :

  Create a parameter grid list for `dtwclust` and `dtw` distance
  functions

- [`descale_by_mase()`](https://ercbk.github.io/ebtools/reference/descale_by_mase.md)
  : Back-transform a MASE-scaled column

- [`dtw_dist_gridsearch()`](https://ercbk.github.io/ebtools/reference/dtw_dist_gridsearch.md)
  :

  Perform a grid search of parameters and `dtwclust` distance functions

- [`prewhitened_ccf()`](https://ercbk.github.io/ebtools/reference/prewhitened_ccf.md)
  : Calculate cross-correlation coefficients for prewhitened time series

- [`scale_by_mase()`](https://ercbk.github.io/ebtools/reference/scale_by_mase.md)
  : Scale a group time series by using a factor derived from the MASE
  error function

- [`test_fable_resids()`](https://ercbk.github.io/ebtools/reference/test_fable_resids.md)
  :

  Autocorrelation test of the residuals of dynamic regression `fable`
  models with various specifications that have been fitted for a
  grouping variable

- [`test_lm_resids()`](https://ercbk.github.io/ebtools/reference/test_lm_resids.md)
  :

  Autocorrelation tests of the residuals of `lm` models with various
  specifications that have been fitted for a grouping variable
