# Print Method for skim_arrow Objects

Provides formatted output for skim_arrow results, displaying summary
statistics organized by variable type in a `skimr`-style format.

## Usage

``` r
# S3 method for class 'skim_arrow'
print(x, ...)
```

## Arguments

- x:

  A skim_arrow object (output from
  [`skim_arrow()`](https://ercbk.github.io/ebtools/reference/skim_arrow.md))

- ...:

  Additional arguments passed to print methods (currently unused)

## Value

Invisibly returns the input object `x`
